# Create the project

```shell
# create folder
mkdir -p ~/ws/github/demos/swift-demos/demo01
cd ~/ws/github/demos/swift-demos/demo01

# use swift package manager to init the project
swift package init --type tool --enable-xctest --name Demo01

# build
swift build
```

# Open it with xcode

```shell
open Package.swift
```
